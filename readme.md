# Qodo SE :: Demos

Demos for Qodo SEs.

## Open source

- `bitwarden-clients`
- `open-webui`
- `screenshot-to-code`

## Sri

- `boring-todo-app`
- `boring-todo-api`

### boring-todo-app

- Next.js app
- Features:
    - Add a todo item
    - List todo items
    - Mark todo item `completed`
    - Delete todo item

### boring-todo-api

- FastAPI server
- Features:
    - Todo item type definition
    - List todo `/items`
